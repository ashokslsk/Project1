# Project 1 & 2
Popular Movies App Stage 1 & 2


## Setup
```sh
 buildTypes.each {
        it.buildConfigField 'String', 'MOVIE_DB_API_KEY', '"YOUR_MOVIE_KEY"'
        it.buildConfigField 'String', 'YOUTUBE_API_KEY', '"YOUR_YOUTUBE_KEY"'
    }
```
